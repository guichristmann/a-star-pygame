Detailed Description

A visualizer for the A* algorithm.

Implements a graphical interface showing the progress of the algorithm while trying to find the optimal path in the world. It also provides text help on the screen and handles keyboard inputs to give commands like start, perform a single step of the search and reset the world.

searchTree An AStartStep object which will be called to execute the algorithm and draw its state on the screen.

squarePixelSize The size of the drawn square/position in pixels.

Iterates through every position in the searchTree world and draws a rectangle according to its value. 0 - Open position. 1 - Obstacle. 2 - Start position for the search. 3 - Goal position. 4 - Searched position.
